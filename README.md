## 🧠 **About the Project**

Welcome to **Tic Tac Toe Mastery**, a beautifully crafted game built in **Python** using **Tkinter** — where logic, design, and artificial intelligence come together!

This project isn’t just a simple game — it’s a hands-on demonstration of how **AI (Minimax Algorithm)** can be implemented to create a challenging and fun experience.
Whether you’re a **beginner learning GUI development** or an **AI enthusiast exploring algorithms**, this project offers valuable insights into both worlds.

🎯 **Goal:** Beat the AI — if you can!

## ⚙️ **How the AI Works**

The **Minimax Algorithm** allows the AI to think like a real player:

1. It evaluates every possible move.
2. Predicts the player’s future responses.
3. Chooses the move that minimizes possible loss.
4. Ensures the AI never makes a random mistake.

Result: 🧠 The AI **never loses**, though you might force a draw if you play perfectly.
